## super()

자식 클래스에서 메서드를 추가로 구현할 때, 부모 클래스의 내용을 사용하고싶으면 super()를 사용할 수 있다.

```python
class BabyClass(ParentClass):
    def method(self, arg):
        super().method(arg) 
```

슈퍼라는 함수로, 부모의 함수를 가져올 수 있다.
